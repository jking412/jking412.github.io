registerGlobalMiddleware()

registerGlobalMiddleware(router *gin.Engine)，在这个函数中我们去注册需要使用的中间件，这里我们使用gin自带的中间件即可，此外setup404Handler(router *gin.Engine)设置了404时的返回结果。

gin.H

可以看到gin.H是一个map[string]interface{}类型，借助于gin.H我们可以快速的构建想要返回的JSON数据
