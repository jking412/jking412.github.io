btsconfig.Initialize()的作用

我们知道Go中的init()函数会在main函数之前被调用，而对于别的包的init()函数而言，它们被调用的时候就是这个包被引用的时候，如果你细心的话就会发现app.go的内容是写在init()函数中的，所以btsconfig.Initialize()的作用就是调用config包下的所有init()函数
